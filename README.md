## 📖 주제: 반려인들이 공감대와 경험을 나눌 수 있는 커뮤니티 플랫폼

### 프로젝트 목적
+ <우리동네 게시판>을 통해 반려동물 관련 시설과 리뷰를 자유롭게 등록하고 공유
+ <멍BTI>나 <멍멍왈왈 게시판> 등을 통해 반려동물에 대한 이해를 높이고 반려인들만의 공감대와 네트워크의 공간을 제공
+ 소켓을 이용한 알람, 채팅 구현 등 1차 프로젝트보다 더 다양한 API 사용
+ 협업중심 프로젝트에 대한 선행 체험
